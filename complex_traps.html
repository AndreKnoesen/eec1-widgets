Complex Number Traps

Four errors that catch everyone once. Work through them before your lab.

Trap 1 of 4

Trap 1

Quadrant Error

A student needs to find the angle of z = −3 + 4j.

θ = tan⁻¹(b/a)

θ = tan⁻¹(4 / −3)

θ = −53.1° ✗

↑ What's wrong here?

The problem

The basic arctangent ignores which quadrant the vector is in. tan⁻¹ only returns values in Q1 or Q4 (between −90° and +90°).

Since the real part is negative and the imaginary part is positive, z is in Q2. Add 180°.

θ = tan⁻¹(|b/a|) = tan⁻¹(4/3) = 53.1°

θ = 180° − 53.1° = 126.9° ✓

Rule: if Re(z) < 0, add 180° to the raw tan⁻¹ result.

Better: use atan2(b, a) which handles all quadrants automatically.

Trap 2

Wrong Form for Addition

🔒 Complete Trap 1 to unlock.

A student needs to add 5∠30° and 3∠60°.

5∠30° + 3∠60°

= (5 + 3)∠(30° + 60°)

= 8∠90° ✗

↑ What's wrong here?

The problem

Polar form only supports direct multiplication and division. You cannot add or subtract angles or magnitudes directly — it simply doesn't work that way geometrically.

Convert to rectangular form first, then add components:

5∠30° = 5·cos(30°) + 5·sin(30°)j = 4.33 + 2.5j

3∠60° = 3·cos(60°) + 3·sin(60°)j = 1.5 + 2.6j

─────────────────────────────────

Sum = 5.83 + 5.1j

= 7.73∠41.2° ✓

Trap 3

Multiplication Cross-Terms Dropped

🔒 Complete Trap 2 to unlock.

A student multiplies (2 + 3j)(1 + 2j).

(2 + 3j)(1 + 2j)

= 2×1 + 3j×2j

= 2 + 6j²

= 2 + 6(−1)

= −4 ✗

↑ What's wrong here?

The problem

Complex multiplication requires FOIL — all four terms. Two cross-terms were dropped.

(2)(1) + (2)(2j) + (3j)(1) + (3j)(2j)

= 2 + 4j + 3j + 6j²

= 2 + 7j + 6(−1)

= −4 + 7j ✓

j² = −1 always. The real and imaginary parts mix through the cross-terms.

Trap 4

Conjugate Misconception

🔒 Complete Trap 3 to unlock.

A student needs the complex conjugate of z = 3 + 4j.

z = 3 + 4j

z* = −3 − 4j ✗

↑ What's wrong here?

The problem

The conjugate only flips the sign of the imaginary part. The real part is unchanged. Flipping both signs gives the negative of z, not the conjugate.

z = 3 + 4j

z* = 3 − 4j ✓

Bonus fact:

z × z* = (3 + 4j)(3 − 4j)

= 9 − 12j + 12j − 16j²

= 9 + 16 = 25 = |z|²

Always real. Always positive. ✓
